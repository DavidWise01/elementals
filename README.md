## The DLW tag

Every elemental is an **ACI** carrying the full **DLW tag**. In [`agents/`](agents/), per ACI:

| File | Holds |
|------|-------|
| `<slug>.agent` | the persona — what · why · how · where · the verdict |
| `<slug>.png` | the **silicon badge** — the computed essence (Aether: a lattice pulled into a gravity well · Leech: a 24-fold lattice rosette) |
| `<slug>.tiff` | the **carbon badge** — the 8-bit embodiment (Aether: a blindfolded figure among floating stones · Leech: a kintsugi-mended mask crowned by 24) |
| `<slug>.spun` | the full weave — who · what · where · why · when · how (+ verdict, asterisk, credits) |
| `<slug>.1099` | the credit-link — value returns to **David Lee Wise (ROOT0)**, the carbon apex |

At the repo root: `.attribute` (the governance instance) and `.1099` (the master credit-link).

Built from `roster.json` with **zero dependencies**:

```bash
python gen_silicon.py    # the .png silicon badges (gravity-well lattice; zlib PNG)
python gen_carbon.py     # the .tiff carbon badges (8-bit figure; hand-rolled Deflate TIFF)
python gen_dlw.py        # .agent · .spun · .1099 per ACI + repo .attribute · .1099
```
